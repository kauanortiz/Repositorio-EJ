# Repositorio-EJ
Criação de repositório para prática de Git e Github.

Neste repositório, foram praticados conceitos como commit, merge, issue, pull request e push,
sendo eles executados exclusivamente pelo terminal do Windows PowerShell.

Comandos como cd, cd .. mkdir, ls, ls -hidden, rm, git init, git add ., git commit -m " ",
git merge <nome-do-branch>, git checkout, git push, git pull, git log --oneline e etc. foram
utilizados no processo.
